# House Price Prediction using Linear Regression

This project is completed as part of the Machine Learning Internship at SkillCraft Technology.

## Objective
To build a Linear Regression model that predicts house prices based on:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
House Prices Advanced Regression Techniques Dataset

## Steps Performed
1. Data Loading
2. Feature Selection
3. Train-Test Split
4. Linear Regression Model Training
5. Prediction
6. Model Evaluation
7. Visualization

## Model Evaluation
- Mean Absolute Error
- R2 Score

## Output
The model predicts house prices and generates a visualization comparing actual vs predicted prices.
